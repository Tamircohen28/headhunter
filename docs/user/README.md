## How HeadHunter fits into your workflow

```
Discover jobs → Scan & score → Apply (tailored CV + cover letter)
     ↓
  Pipeline (Saved → Applied → Phone Screen → Technical → Onsite → Offer → Accepted)
     ↓
  Interview prep (brief + research + mock) → Insights → Negotiate → Follow-up
```

Every step is a slash command. Your data never leaves your machine.
